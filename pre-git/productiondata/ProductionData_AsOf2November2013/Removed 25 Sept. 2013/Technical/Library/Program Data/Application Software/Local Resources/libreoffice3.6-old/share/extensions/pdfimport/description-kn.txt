PDF ಇಂಪೋರ್ಟ್ ಎಕ್ಸ್‍ಟೆನ್ಶನ್ ಎನ್ನುವುದು PDF ದಸ್ತಾವೇಜುಗಳನ್ನು ಆಮದು ಮಾಡಿಕೊಳ್ಳಲು ಮತ್ತು ಮಾರ್ಪಡಿಸಲು ಅವಕಾಶವನ್ನು ನೀಡುತ್ತದೆ.  "PDF/ODF hybrid file" ವಿನ್ಯಾಸದೊಂದಿಗೆ ಅತ್ಯುತ್ತಮವಾದ 100% ರೂಪವಿನ್ಯಾಸದ ನಿಖರತೆಯನ್ನು ಪಡೆಯಲು ಸಾಧ್ಯವಿರುತ್ತದೆ. ಇದನ್ನೂ ಸಹ ಈ ವಿಸ್ತರಣೆಯು ಸಕ್ರಿಯಗೊಳಿಸುತ್ತದೆ. ಒಂದು ಹೈಬ್ರಿಡ್ PDF/ODF ಕಡತವು ಅಡಕಗೊಳಿಸಲಾದ ODF ಆಕರ ಕಡತವನ್ನು ಹೊಂದಿರುವ ಒಂದು PDF ಕಡತವಾಗಿರುತ್ತದೆ. ಹೈಬ್ರಿಡ್ PDF/ODF ಕಡತಗಳನ್ನು ಯಾವುದೆ ರೂಪವಿನ್ಯಾಸ ಬದಲಾವಣೆ ಮಾಡದೆ LibreOffice ನಲ್ಲಿ ODF ಕಡತವಾಗಿ ತೆರೆಯಲಾಗುತ್ತದೆ.
