ផ្នែក​បន្ថែម​កុងសូល​កម្មវិធី​បង្ហាញ ផ្ដល់​ឲ្យ​អ្នក​កាន់តែ​ច្រើន​នូវ​ការ​គ្រប់គ្រង​លើ​ការ​បញ្ចាំង​ស្លាយ​បទ​បង្ហាញ​របស់​អ្នក ដូចជា​សមត្ថភាព​ក្នុង​ការ​មើល​ឃើញ​ស្លាយ​បន្ទាប់ កំណត់ចំណាំ​របស់​ស្លាយ និង​ឧបករណ៍​កំណត់​ពេល​បទ​បង្ហាញ នៅ​ពេល​ដែល​អ្នក​ស្ដាប់​មើល​ឃើញ​តែ​ស្លាយ​បច្ចុប្បន្ន​ប៉ុណ្ណោះ ។
ដើម្បី​ជៀសវាង​ការ​ភាន់ច្រឡំ​ដោយ​សារ​ការ​បង្ហាញ​ធាតុ​ជាច្រើន​របស់​បទ​បង្ហាញ កុងសូល​កម្មវិធី​បង្ហាញ បង្ហាញ​ធាតុ​នៅ​លើ​ទិដ្ឋភាព​ដែល​អាច​ផ្លាស់ប្ដូរ​បាន​យ៉ាង​ងាយស្រួល​ផ្សេង​គ្នា​ចំនួន ៣ ។
