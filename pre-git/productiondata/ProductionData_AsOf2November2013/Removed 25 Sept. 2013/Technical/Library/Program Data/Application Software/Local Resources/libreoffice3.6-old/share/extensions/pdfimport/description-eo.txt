La PDF-Importa kromprogramo ebligas al vi importi kaj modifi PDF-dokumentojn. Plejbonaj rezultoj kun 100% ekzakteco estas atingeblaj per la formato "PDF/ODF-dosiera hibrida dosiero", kiun ĉi tiu kromprogramo ankaŭ ebligas. Hibrida PDF/ODF-dosiero estas PDF-dosiero kiu enhavas enkorpigitan ODF-font-dosieron. Hibridajn PDF/ODF-dosierojn LibreOffice malfermas sen ajna ŝanĝo de aranĝo.
