A extensão Importação de PDF permite-lhe importar e modificar documentos PDF. Os melhores resultados são obtidos se utilizar o formato híbrido "PDF/ODF", que esta extensão também disponibiliza. Um ficheiro híbrido PDF/ODF é um ficheiro PDF que também contém o documento original em ODF. Os ficheiros híbridos PDF/ODF são carregados no LibreOffice sem quaisquer alterações ao esquema.
