வழங்குநர் சாதனம் நீட்டிப்பு, உங்கள் ஸ்லைடு காட்சி விளக்கக்காட்சியின் மீது உங்களுக்கு அதிகப்படியான கட்டுப்பாட்டை வழங்குகிறது, அதைக் கொண்டு நீங்கள் அடுத்து வரும் ஸ்லைடு, ஸ்ளைடு குறிப்புகள் விளக்கக்காட்சி டைமர் ஆகியவற்றையெல்லாம் காண முடியும், ஆனால் பார்வையாளர்களோ நடப்பு விளக்கக்காட்சியை மட்டும் பார்ப்பர்.
அதிக எண்ணிக்கையிலான விளக்கக்காட்சி கூறுகளைக் காண்பித்து குழப்பம் ஏற்படாமல் தவிர்க்க, விளக்கக்காட்சி சாதனம் நீட்டிப்பானது எளிதில் மாற்றத்தக்க மூன்று வெவ்வேறு காட்சிகளில் காண்பிக்கிறது.
