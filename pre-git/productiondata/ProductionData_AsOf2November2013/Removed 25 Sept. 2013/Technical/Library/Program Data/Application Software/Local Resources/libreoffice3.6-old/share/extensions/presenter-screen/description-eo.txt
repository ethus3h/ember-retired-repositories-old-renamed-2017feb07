La kromprogramo Prezentila Konzolo ebligas pli fajnan regadon pri via lumbildoprezento, ekzemple la eblecon vidi la sekvan lumbildon, la notojn pri la lumbildo, kaj prezentilan horloĝon, dum la spektantoj vidas nur la aktualan lumbildon.
Por eviti konfuzon vidigante grandan nombron da prezentaj elementoj, la Prezentila Konzolo vidigas la elementojn per tri malsamaj facile ŝanĝeblaj vidigoj.
