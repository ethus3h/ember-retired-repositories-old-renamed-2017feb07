PDF innflutningsviðbótin gerir kleift að flytja inn og breyta PDF-skjölum. Besta útkoman með 100% nákvæmni framsetningar fæst með því að nota "PDF/ODF blendingsskráasniðið" (hybrid file format), sem einnig er virkjað í þessari viðbót.  PDF/ODF blendingsskrá er PDF-skrá sem inniheldur ígrædda ODF upprunaskrá. PDF/ODF blendingsskrár opnast í LibreOffice sem ODF-skrár án nokkurra útlitsbreytinga.
