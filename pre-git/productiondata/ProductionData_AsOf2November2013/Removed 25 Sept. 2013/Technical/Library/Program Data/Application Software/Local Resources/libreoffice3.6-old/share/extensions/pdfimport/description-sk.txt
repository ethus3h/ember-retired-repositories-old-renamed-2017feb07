Rozšírenie PDF Import umožňuje importovanie a úpravu PDF dokumentov. Najlepšie výsledky so 100% presnosťou rozloženia dokumentu možno dosiahnuť pri použití hybridného formátu "PDF/ODF hybrid file", ktorého použitie toto rozšírenie tiež umožňuje. A hybridný súbor PDF/ODF je PDF súbor, ktorý obsahuje vloženú zdrojovú ODF verziu. Hybridné súbory PDF/ODF sa v LibreOffice otvoria ako ODF súbory bez akejkoľvek zmeny formátovania.
