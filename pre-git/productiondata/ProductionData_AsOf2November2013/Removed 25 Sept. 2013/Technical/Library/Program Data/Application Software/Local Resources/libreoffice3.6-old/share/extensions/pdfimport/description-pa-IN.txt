PDF ਇੰਪੋਰਟ ਐਕਸਟੈਂਸ਼ਨ ਤੁਹਾਨੂੰ PDF ਡੌਕੂਮੈਂਟ ਇੰਪੋਰਟ ਅਤੇ ਸੋਧਣ ਵਿੱਚ ਮੱਦਦ ਕਰਦੀ ਹੈ। 100% ਲੇਆਊਟ ਸ਼ੁੱਧੀ ਨਾਲ ਵਧੀਆ ਨਤੀਜਾ "PDF/ODF ਹਾਈਬਰਿੱਡ ਫਾਇਲ" ਫਾਰਮੈਟ ਨਾਲ ਪ੍ਰਾਪਤ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ। ਜਿਸਨੂੰ ਇਹ ਐਕਸਟੈਂਸ਼ਨ ਵੀ ਯੋਗ ਕਰਦੀ ਹੈ। ਇੱਕ ਹਾਈਬਰਿੱਡ PDF/ODF ਫਾਇਲ ਇੱਕ PDF ਫਾਇਲ ਹੈ ਜਿਸ ਵਿੱਚ ਇੱਕ ODF ਸੋਰਸ ਫਾਇਲ ਸ਼ਾਮਿਲ ਕੀਤੀ ਗਈ ਹੈ। ਹਾਈਬਰਿੱਡ PDF/ODF ਫਾਇਲਾਂ ਨੂੰ LibreOffice ਵਿੱਚ ਇੱਕ ODF ਫਾਇਲ ਵਾਂਗ ਬਿਨਾਂ ਲੇਆਊਟ ਤਬਦੀਲੀ ਖੋਲਿਆ ਜਾਵੇਗਾ।
