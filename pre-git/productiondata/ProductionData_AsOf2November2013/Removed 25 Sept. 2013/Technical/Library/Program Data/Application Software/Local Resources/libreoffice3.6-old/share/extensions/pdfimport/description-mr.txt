PDF इम्पोर्ट एक्सटेंशन तुम्हाला PDF दस्तऐवज आयात व संपादित करायला परवानगी देते. "PDF/ODF हायब्रिड फाइल" रूपणसह 100% लेआउट मांडणीसह प्राप्त करणे शक्य आहे, जे हे एक्सटेंशन देखिल सुरू करते. हायब्रिड PDF/ODF फाइल ति PDF फइल असते ज्यामध्ये एम्बेड केलेले ODF सोअर्स फाइल असते. हायब्रिड PDF/ODF फाइलला LibreOffice मध्ये कोणतिहि लेआउट मांडणीविना ODF फाइल म्हणून उघडले जाते.
