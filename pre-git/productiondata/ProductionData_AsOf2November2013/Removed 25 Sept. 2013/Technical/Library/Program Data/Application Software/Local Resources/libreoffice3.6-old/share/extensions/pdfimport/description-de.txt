Die PDF-Import Extension ermöglicht es Ihnen, PDF-Dokumenten zu importieren und zu bearbeiten. Die besten Ergebnisse mit 100% Layout-Genauigkeit können mit dem "PDF/ODF Hybridformat" erreicht werden, welches diese Erweiterung ebenfalls bereitstellt. Eine Hybrid-PDF/ODF-Datei ist eine PDF-Datei, die eine eingebettete ODF-Datei enthält. Hybrid PDF/ODF-Dateien werden in LibreOffice als ODF-Datei ohne Änderungen am Layout geöffnet.
