Η επέκταση εισαγωγής PDF επιτρέπει την εισαγωγή και τροποποίηση εγγράφων PDF. Άριστα αποτελέσματα με 100% ακρίβεια διάταξης μπορούν να επιτευχθούν με τον τύπο "αρχείο υβριδίου PDF/ODF", τον οποίο αυτή η επέκταση επίσης ενεργοποιεί. Ένα υβριδικό αρχείο PDF/ODF είναι ένα αρχείο PDF που περιέχει ένα ενσωματωμένο πηγαίο αρχείο ODF.  Τα υβριδικά αρχεία  PDF/ODF θα ανοιχτούν στο LibreOffice ως ένα αρχείο ODF χωρίς αλλαγές διάταξης.
