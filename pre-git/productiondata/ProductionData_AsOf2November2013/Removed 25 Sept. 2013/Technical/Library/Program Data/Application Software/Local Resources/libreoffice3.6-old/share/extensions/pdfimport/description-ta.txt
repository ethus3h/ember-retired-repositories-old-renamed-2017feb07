PDF இறக்குமதி நீட்டிப்பைக் கொண்டு நீங்கள் PDF ஆவணங்களை இறக்குமதி செய்யவும் ஆவற்றில் மாற்றங்கல் செய்யவும் முடியும். "PDF/ODF hybridfile" கோப்பு வடிவத்துடன் பயன்படுத்தினால் இதைக் கொண்டு 100% தளவமைப்பு துல்லியத்தைப் பெறலாம், இந்த நீட்டிப்பு இந்த கோப்பு வடிவத்தையும் செயல்படுத்துகிறது. ஹைப்ரிட் PDF/ODF கோப்பு என்பது உட்பொதிக்கப்பட்ட ODF மூலக் கோப்பைக் கொண்ட PDF கோப்பாகும். ஹைப்ரிட் PDF/ODF கோப்புகள் LibreOffice இல் தளவமைப்பு மாற்றங்கள் ஏதுமில்லாத ODF கோப்பாக திறக்கப்படும்.
