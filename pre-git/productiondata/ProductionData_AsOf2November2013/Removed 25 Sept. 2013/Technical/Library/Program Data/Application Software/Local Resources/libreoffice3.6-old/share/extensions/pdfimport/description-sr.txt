Додатак за увоз ПДФ документа омогућава да отворите и мењате ПДФ документе. Најбољи резултати уз 100% тачности постижу се ако користите „хибридни ПДФ/ОДФ формат“ који је омогућен уз овај додатак. Хибридна датотека је ПДФ датотека која у себи садржи цео изворни ОДФ документ. Можете је гледати у стандардном ПДФ прегледачу, док ће Либреофис отворити ОДФ за измену без икаквих губитака.
