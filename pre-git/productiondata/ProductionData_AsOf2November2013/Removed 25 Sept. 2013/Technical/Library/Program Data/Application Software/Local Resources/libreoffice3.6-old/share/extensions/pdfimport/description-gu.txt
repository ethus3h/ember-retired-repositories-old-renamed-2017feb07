PDF આયાત ઍક્સટેન્શન આયાત કરવા માટે તમને પરવાનગી આપે છે અને PDF દસ્તાવેજોને બદલો. 100% લેઆઉટ ચોક્કસતા સાથે સારાં પરિણામો "PDF/ODF હાયબ્રિડફાઇલ" બંધારણ સાથે પ્રાપ્ત કરી શકાય છે, કે જે આ ઍક્સટેન્શન પણ સક્રિય થાય છે. હાઇબ્રિડ PDF/ODF ફાઇલ એ PDF ફાઇલ છે કે જે ઍમ્બેડેડ ODF સ્ત્રોત ફાઇલને સમાવે છે. હાઇબ્રિડ PDF/ODF ફાઇલો કોઇપણ લેઆઉટ ફેરફારો વગર ODF ફાઇલ તરીકે LibreOffice માં ખૂલ્લુ હશે.
