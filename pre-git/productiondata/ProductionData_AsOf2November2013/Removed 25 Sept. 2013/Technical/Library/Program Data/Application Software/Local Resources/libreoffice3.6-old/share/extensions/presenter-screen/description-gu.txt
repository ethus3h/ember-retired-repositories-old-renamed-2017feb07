રજૂઆત કન્સોલ ઍક્સટેન્શન એ તમારી તકતી પૂર્વદર્શન રજૂઆત પર વધારે નિયંત્રણ પૂરુ પાડે છે, જેમ કે આવતી તકતીને જોવાં સક્ષમતા, તકતી નોંધો, અને રજૂઆત ટાઇમર જ્યારે પ્રેક્ષકો વર્તમાન તકતીને જુએ.
વિશાળ સંખ્યામાં રજૂઆત ઘટકોને દર્શાવાથી મૂંઝવણોને અવગણવા માટે, રજૂઆત કન્સોલ ત્રણ વિવિધ સરળ રીતે બદલી શકાય તેવાં દૃશ્યો પર ઘટકોને દર્શાવે છે.
