A extensão "Importação de PDF" permite importar e modificar documentos PDF. Os melhores resultados são obtidos ao utilizar o formato híbrido "PDF/ODF", que esta extensão também disponibiliza. Um arquivo híbrido "PDF/ODF", é um arquivo PDF que também contém o documento fonte em formato ODF. Os arquivos híbridos PDF/ODF, são abertos pelo LibreOffice sem quaisquer alterações no layout.
