Η επέκταση κονσόλας παρουσίασης παρέχει περισσότερο έλεγχο στην προβολή παρουσίασης διαφανειών, όπως η ικανότητα να δει την επερχόμενη διαφάνεια, τις σημειώσεις διαφάνειας και ένα χρονόμετρο παρουσίασης, ενώ το ακροατήριο βλέπει μόνο την τρέχουσα διαφάνεια.
Για αποφυγή συγχύσεων με την εμφάνιση ενός μεγάλου αριθμού στοιχείων παρουσίασης, η κονσόλα παρουσίασης εμφανίζει τα στοιχεία σε τρεις διαφορετικές εύκολα εναλλάξιμες προβολές.
