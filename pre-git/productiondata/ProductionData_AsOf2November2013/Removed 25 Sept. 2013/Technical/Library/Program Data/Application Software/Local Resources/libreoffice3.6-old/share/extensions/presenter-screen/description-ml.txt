സ്ലൈഡ് ഷോ പ്രസന്റേഷനു് കൂടുതല്‍ നിയന്ത്രണം പ്രസന്റര്‍ കണ്‍സോള്‍ എക്സ്റ്റെന്‍ഷന്‍ ലഭ്യമാക്കുന്നു. അടുത്ത സ്ലൈഡുകള്‍, സ്ലൈഡ് കുറിപ്പുകള്‍, പ്രസന്റേഷന്‍ ടൈമര്‍ എന്നിവ ഇതു് ലഭ്യമാക്കുന്നു, അതേസമയം ദര്‍ശകര്‍ക്കു് നിലവിലുള്ള സ്ലൈഡ് മാത്രമേ കാണുവാന്‍ സാധിയ്ക്കൂ.
പ്രസന്റേഷന്‍ എലമെന്റുകളുടെ ഏറ്റവും കൂടിയ നംബറുകള്‍ കാണിയ്ക്കുന്നതു് ഒഴിവാക്കുന്നതിനായി, പ്രസന്റര്‍ കണ്‍സോള്‍ പലതരത്തിലുള്ള മൂന്നും കാഴ്ചകള്‍ ലഭ്യമാക്കുന്നു.
