Разширението Presenter Console ви предоставя повече контрол върху прожектирането на презентацията, например възможност да виждате следващия кадър, бележките към кадъра и таймер, докато публиката вижда само самия кадър.
За да се избегне объркване при показване на голям брой елементи на презентацията, Presenter Console показва елементите в три различни лесно заменящи се изгледа.
