Utvidelsen PDF-importering gjør det mulig å importere og redigere PDF-dokumenter. De beste resultatene med tanke på 100 % utseendenøyaktighet kan oppnås ved å bruke filformatet «PDF/ISO-hybrid», som denne utvidelsen slår på. En PDF/ISO-hybridfil er en PDF-fil som inneholder en innebygd ODF-kildefil. Slike hybridfiler vil åpnes i LibreOffice som en ODF-fil, uten endringer i utforminga.
