PDF దిగుమతి పొడిగింపు మిమ్ములను PDF పత్రములను దిగుమతి చేయుటకు మరియు సవరించుటకు అనుమతించును. 100% నమూనా ఖచ్చితత్వంతో మంచి ఫలితాలు  "PDF/ODF hybrid file" ఫార్మాట్‌తో పొందవచ్చు, ఈ పొడిగింత యిది కూడా చేతనంచేయును. హైబ్రిడ్ PDF/ODF ఫైలు అనేది PDF ఫైలు అది యెంబెడెడ్ ODF మూలపు ఫైలును కలిగివుంటుంది. హైబ్రిడ్ PDF/ODF ఫైళ్ళు LibreOffice నందు ODF ఫైలువలె తెరువబడును యెటువంటి లేఅవుట్ మార్పులు లేకుండా.
