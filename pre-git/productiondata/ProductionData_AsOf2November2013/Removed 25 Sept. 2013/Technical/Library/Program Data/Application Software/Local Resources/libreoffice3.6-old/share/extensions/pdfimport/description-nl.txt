De extensie PDF importeren stelt u in staat om PDF documenten te importeren en te wijzigen. Een goed resultaat met een 100% accurate opmaak kan worden verkregen met het "PDF/ODF hybride bestandsformaat", die deze extensie ook inschakelt. Een hybride PDF/ODF bestand is een PDF bestand waarin een ODF bronbestand is ingebed. Hybride PDF/ODF bestanden zullen in LibreOffice als een ODF bestand worden geopend zonder enige wijzigingen in de opmaak.
