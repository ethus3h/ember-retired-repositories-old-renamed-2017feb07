ਪੇਸ਼ਕਾਰੀ ਕੰਸੋਲ ਐਕਸਟੈਂਸ਼ਨ ਤੁਹਾਡੀ ਸਲਾਈਡ ਸ਼ੋ ਪੇਸ਼ਕਾਰੀ ਉੱਪਰ ਵਧੀਆ ਕੰਟਰੋਲ ਦਿੰਦੀ ਹੈ, ਜਿਵੇਂ ਆਉਣ ਵਾਲੀ ਸਲਾਈਂਡ ਬਾਰੇ ਜਾਣਕਾਰੀ, ਸਲਾਈਡ ਸੂਚਨਾ, ਅਤੇ ਇੱਕ ਪੇਸ਼ਕਾਰੀ ਟਾਈਮਰ ਜਿੱਥੇ ਸਰੋਤੇ ਸਿਰਫ ਮੌਜੂਦਾ ਸਲਾਈਡ ਵੇਖਦੇ ਹਨ।
ਪੇਸ਼ਕਾਰੀ ਐਲੀਮੈਂਟਾਂ ਦੀ ਵੱਡੀ ਗਿਣਤੀ ਵੇਖਾਉਣ ਵਾਲੇ ਭੰਬਲਭੂਸੇ ਤੋਂ ਬਚਣ ਲਈ, ਪੇਸ਼ਕਾਰੀ ਕੰਸੋਲ ਤਿੰਨ ਵੱਖ-ਵੱਖ ਅਸਾਨੀ ਨਾਲ ਤਬਦੀਲ ਹੋਣ ਯੋਗ ਦ੍ਰਿਸ਼ਾਂ ਨੂੰ ਵੇਖਾਉਂਦਾ ਹੈ।
