Прыстаўка 'Імпарт PDF' дазваляе імпартаваць і правіць дакументы PDF. Найлепшыя вынікі са 100% перадачай выкладу дасягаюцца з фарматам "PDF/ODF гібрыдны файл", які даступны і ў прыстаўцы. Гібрыдны файл PDF/ODF гэта файл PDF, у якім зашыты яго вытокавы файл ODF. Гібрыдныя файлы PDF/ODF адкрываюцца ў LibreOffice як файлы ODF без змяненняў у выкладзе.
