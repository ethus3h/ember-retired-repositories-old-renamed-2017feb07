प्रस्तुति कंसोल विस्तार आपको अपने स्लाइड शो प्रस्तुत पर अधिक नियंत्रण देता है, जैसे कि आनेवाले स्लाइड को देखने की क्षमता, स्लाइड नोड, और प्रस्तुति टाइमप जहाँ श्रोता केवल मौजूदा स्लाइड देख पाते हैं.
प्रस्तुति तत्व की बड़ी संख्या को दिखाते हुए भ्रम से हटने के लिए, प्रस्तुति कंसोल तीन विभिन्न बदलने वाले तरीके पर दिखाता है.
