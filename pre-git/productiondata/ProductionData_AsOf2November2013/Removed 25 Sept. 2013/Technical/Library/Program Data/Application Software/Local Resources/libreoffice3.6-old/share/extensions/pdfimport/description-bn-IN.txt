PDF ইম্পোর্ট এক্সটেনশনের মাধ্যমে PDF নথিপত্র ইম্পোর্ট ও পরিবর্তন করা যাবে। এই এক্সটেনশনে উপলব্ধ  "PDF/ODF hybrid file" বিন্যাস দ্বারা ১০০% যথার্থতা প্রাপ্ত করা সম্ভব হবে। হাইব্রিড PDF/ODF ফাইল মূলত একটি PDF ফাইল যার মধ্যে ODF সোর্স ফাইল এমবেড করা থাকে। হাইব্রিড PDF/ODF ফাইলগুলি LibreOffice-র মধ্যে কোনো বিন্যাস পরিবর্তন না করে ODF ফাইল রূপে প্রদর্শন করা হবে।
