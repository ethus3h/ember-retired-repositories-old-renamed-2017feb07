പിഡിഎഫ് രേഖകള്‍ ഇംപോര്‍ട്ട് ചെയ്തു് അവയില്‍ മാറ്റം വരുത്തുന്നതിനുള്ള പ്രയോഗമാണു് പിഡിഎഫ് ഇംപോര്‍ട്ട് എക്സ്റ്റെന്‍ഷന്‍. "PDF/ODF ഹൈബ്രിഡ് ഫയല്‍" എന്ന ശൈലിയില്‍ 100% ലേയൌട്ട് ലഭ്യമാകുന്നു, ഈ എക്സ്റ്റെന്‍ഷനും ഇതു് സജ്ജമാക്കുന്നു. PDF/ODF ഹൈബ്രിഡ് ഫയലില്‍ എംബഡ് ചെയ്ത ഒഡിഎഫ് സോഴ്സ് ഫയല്‍ ലഭ്യമാകുന്നു. PDF/ODF ഹൈബ്രിഡ് ഫയലുകള്‍ ലിബര്‍ഓഫീസില്‍ ഒഡിഎഫ് ഫയലുകളായി തുറക്കപ്പെടുന്നു.
