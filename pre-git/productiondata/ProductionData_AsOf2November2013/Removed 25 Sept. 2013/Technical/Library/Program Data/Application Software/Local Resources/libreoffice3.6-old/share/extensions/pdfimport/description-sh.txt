Dodatak za uvoz PDF dokumenta omogućava da otvorite i menjate PDF dokumente. Najbolji rezultati uz 100% tačnosti postižu se ako koristite „hibridni PDF/ODF format“ koji je omogućen uz ovaj dodatak. Hibridna datoteka je PDF datoteka koja u sebi sadrži ceo izvorni ODF dokument. Možete je gledati u standardnom PDF pregledaču, dok će Libreofis otvoriti ODF za izmenu bez ikakvih gubitaka.
