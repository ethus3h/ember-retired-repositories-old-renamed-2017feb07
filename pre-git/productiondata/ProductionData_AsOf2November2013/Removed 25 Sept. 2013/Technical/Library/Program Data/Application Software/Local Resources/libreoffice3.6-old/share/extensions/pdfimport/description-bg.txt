Разширението PDF Import ви позволява да импортирате и променяте документи във формат PDF. Най-добри резултати със 100-процентова точност на оформлението се получават с хибридния формат „PDF/ODF файл“, също реализиран в това разширение. Хибридният PDF/ODF файл е PDF файл, в който е вграден ODF файл – източник. Хибридните PDF/ODF файлове се отварят в LibreOffice като ODF файлове, без никакви промени в оформлението.
