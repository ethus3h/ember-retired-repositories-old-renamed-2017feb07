ההרחבה ליבוא PDF מאפשרת לך ליבוא ולשנות מסמכי PDF. ניתן להגיע לתוצאות מיטביות עם 100% דיוק במתאר על ידי בחירה במבנה "קובץ מעורב PDF/ODF" בו ניתן להשתמש באמצעות הרחבה זו. קובץ מעורב PDF/ODF הוא קובץ PDF המכיל קובץ מקור ODF מוטמע. קובצי PDF/ODF מעורבים ניתנים לפתיחה באמצעות LibreOffice כמו קובץ ODF ללא שינויי מתווה כלשהם.
