Extensia PDF Import permite importarea și modificarea documentelor PDF. Rezultate 100% în acuritate puteți obține cu utilizarea formatului de fișiere PDF/ODF hibrid. Această extensie suportă acest format. Formatul PDF/ODF hibrid este un fișier PDF care conține și sursa originală ODF. Fișierele PDF/ODF hibrid, LibreOffice le deschide ca fișiere ODF, din acest motiv nici o infomație de formatare nu dispare.
