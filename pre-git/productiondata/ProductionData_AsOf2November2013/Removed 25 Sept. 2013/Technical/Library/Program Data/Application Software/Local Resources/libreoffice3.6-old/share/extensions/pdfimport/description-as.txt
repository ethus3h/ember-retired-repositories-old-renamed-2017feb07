PDF ইমপোৰ্ট সম্প্ৰসাৰণে আপোনাক PDF দস্তাবেজসমূহ ইমপোৰ্ট আৰু পৰিৱৰ্তন কৰাৰ অনুমতি দিয়ে। ১০০% বিন্যাস সঠিকতাৰ সৈতে উত্তম ফলাফল "PDF/ODF hybrid file" বিন্যাসৰ সৈতে প্ৰাপ্ত কৰিব পাৰি, যি এই সম্প্ৰসাৰণে সামৰ্থবান কৰে। এটা হাইব্ৰিড PDF/ODF নথিপত্ৰ এটা PDF নথিপত্ৰ যি এটা অন্তৰ্ভুক্ত ODF উৎস নথিপত্ৰ অন্তৰ্ভুক্ত কৰে। হাইব্ৰিড PDF/ODF নথিপত্ৰসমূহক LibreOffice ত কোনো বিন্যাস পৰিৱৰ্তন নহোৱাকৈ এটা ODF নথিপত্ৰ হিচাপে খোলা হব।
