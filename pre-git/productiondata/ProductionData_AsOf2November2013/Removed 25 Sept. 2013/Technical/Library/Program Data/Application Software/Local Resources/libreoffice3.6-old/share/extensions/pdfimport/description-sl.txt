Razširitev Uvoznik dokumentov PDF omogoča uvoz in spreminjanje dokumentov PDF. Najboljše rezultate s popolno natančnostjo lahko dosežete z zapisom "hibridnih datotek PDF/ODF", ki ga razširitev tudi omogoča. Hibridna datoteka PDF/ODF je datoteka PDF, ki vsebuje vdelano izvorno datoteko ODF. Hibridne datoteke PDF/ODF se odpirajo v LibreOffice kot datoteke ODF brez vsakršnih sprememb v postavitvi.
