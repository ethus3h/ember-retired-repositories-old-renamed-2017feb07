प्रेजेंटर कंसोल स्लाइड शो प्रस्तुतिकरणकरीता अधिक नियंत्रण पुरवण्यासाठी एक्सटेंशन पुरवते, जसे कि पुढील स्लाइड, स्लाइड नोटस्, व प्रेजेंटेशन टाइमर पहाण्याची क्षमता, जेव्हा श्रोता फक्त सध्याची स्लाइड पहाते.
प्रस्तुतिकरण घटकांची मोठी संख्या दाखवून गोंधळ टाळण्यासाठी, प्रेजेंटर कंसोल तीन वेगळे सोपेरित्या अवलोकन बदलण्यासाठी घटके दाखवतो.
