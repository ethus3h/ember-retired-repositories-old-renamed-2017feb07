Расширение Presenter Console Extension управляет показом презентации, позволяя докладчику видеть следующие слайды, примечания и таймер, в то время как зрители видят только текущий слайд.
Во избежание путаницы при показе большого числа элементов презентации Presenter Console отображает элементы тремя различными способами.
