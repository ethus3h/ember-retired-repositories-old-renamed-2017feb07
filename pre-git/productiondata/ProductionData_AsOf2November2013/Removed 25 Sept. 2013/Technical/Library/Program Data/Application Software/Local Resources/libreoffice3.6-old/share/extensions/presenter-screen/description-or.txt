ପ୍ରଚଳିତ କୋନସଲ ଅନୁଲଗ୍ନ ଆପଣଙ୍କ ସ୍ଲାଇଡ ପ୍ରଦର୍ଶନ ଉପରେ  ଅଧିକ ନିୟନ୍ତ୍ରଣ ପ୍ରଦାନ କରିଥାଏ, ଯେପରିକି ଆସୁଥିବା ସ୍ଲାଇଡକୁ ଦେଖିବାର କ୍ଷମତା, ସ୍ଲାଇଡ ଟିପ୍ପଣୀ, ଏବଂ ଏକ ଉପସ୍ଥାପନା ସମୟ ମାପକ ଯେଉଁଠିକି ଦର୍ଶକ କେବଳ ପ୍ରଚଳିତ ସ୍ଲାଇଡକୁ ଦେଖିପାରନ୍ତି।
ସନ୍ଦେହ ଦୂର କରିବା ପାଇଁ ଏକ ବଡ଼ ଆକାରର ଉପସ୍ଥାପନା ଉପାଦାନ ପ୍ରଦର୍ଶନ କରି, ଉପସ୍ଥାପକ କୋନସଲ ଉପାଦାନଗୁଡ଼ିକୁ ତିନୋଟି ଭିନ୍ନ ଉପାୟରେ ଦର୍ଶାଇଥାଏ।
