Installation (click the logo above to return home)

To install Wreathe:

  1. Download the current version using the link on the home page, and use the cryptographic checksum to verify that the file is not corrupted.
  2. Verify that your computer meets the system requirements (although some of these can be ignored if you are technically knowledegable):
    • IBM-compatible i686 or greater CPU, 2 GHz or faster
    • Hard drive or SSD (MBR style partition table recommended), with 14 GB available space
    • DVD-R burner
    • Compatible video card with 3D acceleration (known to work well: ATI Radeon 9600 with 128 MB video memory)
    • Color display, LCD 1280x1024 recommended
    • 720 MB RAM

  3. Burn it to a DVD. (It is recommended to use slow burning speeds to prevent data corruption, and then to verify the burned DVD afterwards.) (There are ways around using a DVD if you know how.)
  4. Boot your computer from the DVD.
  5. Set the permissions of the installer icon in ~/Technical/Desktop to executable (chmod +x).
  6. Run the installer and follow the instructions it gives. WARNING: Formatting a partition or drive will erase ALL the data on the partition or drive being formatted. Having a complete and verified backup of all your data is strongly recommended.
  7. Restart the computer not using the DVD after the installation is complete.
