Wreathe is a Linux distribution targeted at intermediate to advanced desktop users. It contains a diverse and useful set of preinstalled application software, enabling it to be effective out-of-the-box in a large number of fields. The goals of Wreathe are to provide a stable, fast system requiring minimal configuration and learning. It is a direct descendent of both the Ubuntu Linux distribution and the FMOS concept systems. The radically rethought user interface provides a very intuitive and powerful method for a variety of desktop computing applications, while still providing ease of access to seemingly endless configuration options and lower-level system functionality. Version 4, "Blossom", adds a number of features and enhancements to Wreathe that should please old users and newcomers alike.

Of course, it's all free.
